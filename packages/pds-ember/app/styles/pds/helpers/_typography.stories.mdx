# Typography Helpers

## Body Copy
Default body copy styles are applied to the `<body>` element, but you can apply
one of the following CSS helper classes to alter copy size.

- `.pds--body-lg`
- `.pds--body-sm`

```html
<p class="pds--body-lg">Large Body Text</p>
<p>Default Body Text</p>
<p class="pds--body-sm">Small Body Text</p>
```
